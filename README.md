# Cheese Out!

This game is still on it's early phase, please come check later! 

Protect your cheese from the eager mices! In the alternative of the popular game "Breakout" you play against someone to eat their cheese and protect yours from them. 

## Deploy on Itch.io
Go try the game here! : [Itch.io](https://spykielucilius.itch.io/cheese-out)
## Features
* **Local Multiplayer:** Play against a friend on the same keyboard.
* **VS Computer:** Play against the computer.
